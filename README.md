Welcome to the **Tourist Management System** ‚Äî a C++ console application that simulates a travel agency's operations. It allows you to manage tours, book destinations, collect tourist details, and display special summer offers with a user-friendly CLI.

---

##  Features

‚úÖ **Login System with Validation**
- Username: exactly 5 lowercase letters  
- Password: exactly 8 digits

‚úÖ **Tour Management**
- Add new tours  
- Update existing bookings  
- View all tours with total revenue  
- Delete tours by ID

‚úÖ **Tourist Booking System**
- Book from predefined tours  
- Enter tourist info and confirm payment  
- Automatic tour creation and data association

‚úÖ **Available Destinations**
- View top Indian travel destinations with hotel and itinerary info

‚úÖ **Special Summer Offers**
- Exclusive limited-time summer destinations with perks
